# ai.md

This repo is owned by ecstacys and was used to test Kiro, an AI-powered development environment.

Kiro is a terminal agent that can inspect and modify GitHub repositories — reading and writing files, creating branches, opening pull requests, filing issues, and searching the web to research anything needed along the way. It has no shell access and works entirely through the GitHub API.

The `meow.md` file was created during that testing session.
